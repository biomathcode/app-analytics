# App analytica Powered By expert.ai API

Turns [google-play-scraper](https://github.com/facundoolano/google-play-scraper/) into a RESTful API.

Set up

```
git clone repo
cd repo
npm install
cd client
npm install 
```

Running the application

Start Server

```
npm start
```

Start Client
```
cd client
npm start

```
